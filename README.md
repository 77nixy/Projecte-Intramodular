# Projecte Intermodular — NEXUS

Foro gaming competitivo desarrollado como Projecte Intermodular.

**Autor:** Hugo Reche
**Curs acadèmic:** 2025–2026

---

## Estructura del repositori

```
Projecte-Intramodular/
├── documentacio/           Memòria i Kanban + control d'hores
└── gaming-foro-v2/         Codi font del foro (PHP + MySQL + XAMPP)
```

## Contingut

### `documentacio/`
- `NEXUS_Memoria.pdf` — Memòria tècnica completa del projecte.
- `NEXUS_Kanban_y_Horas.pdf` — Tauler Kanban i registre d'hores dedicades a cada tasca.

### `gaming-foro-v2/`
Codi font del foro NEXUS. Aplicació web PHP sobre XAMPP (Apache + MySQL).
Consultar el `README.txt` dins de la carpeta per a instal·lació, credencials i estructura interna.

## Tecnologies

- PHP 8+
- MySQL / MariaDB
- HTML, CSS, JavaScript
- XAMPP (entorn local)

## Com executar el projecte

1. Copiar la carpeta `gaming-foro-v2/` dins de `C:\xampp\htdocs\`.
2. Executar `ARRANCAR.bat` com a administrador.
3. Obrir `http://localhost/gaming-foro-v2/` al navegador.

Veure `gaming-foro-v2/README.txt` per a més detalls.
